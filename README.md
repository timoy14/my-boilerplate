# Alrapeh App (Dashboard + Api)

## Features

-   Laravel 7
-   API Authentication
-   GraphQL + GraphiQl
-   Rukuda Theme

## Installation

-   `Open Terminal`
-   `git clonehttps://github.com/Abdullah9/alrapeh_Api`
-   `composer install`
-   `npm install && npm run dev`
-   `php artisan migrate && php artisan db:seed`
-   `php artisan serve`
